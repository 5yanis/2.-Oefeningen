h element: Voor een titel hoe groter het getal hoe kleiner de lettertype
p element: voor onder je titel een tekst zoals bij een artikel
a element: van een tekst iets vetgrdukt maken, importeer je dit in een img scr dan maak je je foto klikbaar naar een link
ol element: een element voor ook een lijst, alleen wordt hier genummerd volgorde is hier van belang!
target: als je je link van een a element naar een niewe tab wil brengen dan vul je _blank in voor dit te doen.
img scr: element voor het importeren van een foto alt= hoort voor als de foto niet inlaad de foto te beschrijven
main: je belangrijke deel van je code, hoofdinfo navigatie hoort er niet bij
section: zoals je hoort groepering van een code voor een overzicht.
ul element: voor een lijst, volgorde hoort er niet bij de elementen bij de lijst is <il> ...</il>
html: hier geef je weer dat je ermee werkt.
figure element: als je een image importeert is het handig om een overzicht ervan te maken. Daarvoor komt de figure element van pas!
figcaption: de tekst dat onder je foto zal komen
em element: om vb in je p element iets op te laten vallen. (Ander lettertype)
strong element: vetgrdukt teskt, opvallen dus!
form element: om info op te halen van de user met de attributen action kies je waar die data heen gaat
input: is wel duidelijk, een deel van form, form is het geheel van alles om data in te voeren/versturen.
type attribuut: het is een attribuut voor een type input en met naam assign je de value van de input aan je keuze. (Radio voor meerkeuze)
placeholder attribuut: Het is een attribuut voor dat de user in de typbox kan zien wat er verwacht word dat erin komt. VB. username, password.
required: maakt de input verplicht.
label element: zorgt ervoor dat de placeholder attr blijft na klikken op het input veldje.
fieldset element: groepeert zoals section alleen meer specifiek vb. een groep en daarbinnen subgroepen. Er komt ook een visual box rondom je radio input
legend element: hetzeflde als figure alleen voor tekst input i.p.v foto captions.

Tip: geef je label input dezelfde naam om het zo te maken dat je maar 1 kan selecteren.

nest: houd in dat je een attr of element in een ander stopt.
